Hide the WordPress admin bar on the frontend for selected user roles

== Description ==
A simple plugin to hide the WordPress admin bar on the frontend for selected user roles.

== Installation ==
1. Upload the `hide-admin-bar` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings > Hide Admin Bar to configure the plugin.

== Frequently Asked Questions ==

= How do I configure which roles are affected? =
Go to Settings > Hide Admin Bar and enter the roles separated by commas.
